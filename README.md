# 🎨 Neural Style Transfer using AdaIN

A Flask-based web application that performs **Neural Style Transfer** using **Adaptive Instance Normalization (AdaIN)**. Upload a content image and a style image, adjust the style strength, and generate artistic images instantly.

## 🚀 Features
- Upload Content & Style Images
- Adjustable Style Strength
- Real-time Style Transfer
- Download Stylized Image
- Modern Responsive UI

## 🛠 Tech Stack
- Python
- Flask
- PyTorch
- Torchvision
- HTML, CSS, JavaScript
- Bootstrap
